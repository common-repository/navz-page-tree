== Description ==
navz Page Tree Structures the pages in WordPress for more robust page edit, sorting and managing. It\'s more clean you know. You can use this plugin to create multiple hierarchy pages for large websites. navz Page Tree also enables you to add multiple pages at once making which saves you a lot of time when create individual pages.

== Installation ==
1. Upload the plugin folder \'navz-page-tree\' to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \"Plugins\" menu in WordPress
3. Done!
